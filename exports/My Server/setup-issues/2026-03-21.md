### 08:42 - @Soumyadip (Divinesoumyadip-git)

The error `Could NOT find GTest` means CMake can't find the Google Test library on your Mac. Run `brew install googletest` to install it. After that, CMake should be able to find it automatically.  If it still fails, you may need to tell CMake where to look by adding `-DGTEST_ROOT=$(brew --prefix googletest)` to your cmake command.
