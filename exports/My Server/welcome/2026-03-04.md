### 21:26 - @Eder Monteiro

Sure, here it is:
1. CUGR directory: https://github.com/The-OpenROAD-Project/OpenROAD/tree/master/src/grt/src/cugr. It has all the source code of the tool.
2. CUGR.cpp: https://github.com/The-OpenROAD-Project/OpenROAD/blob/master/src/grt/src/cugr/src/CUGR.cpp. The main file of the tool. You could start looking at the CUGR::route() function to understand how its flow works.
3. Design.cpp: https://github.com/The-OpenROAD-Project/OpenROAD/blob/master/src/grt/src/cugr/src/Design.cpp. This is where the design data is loaded into the tool, and probably where the most of the updates to support incremental will be made
